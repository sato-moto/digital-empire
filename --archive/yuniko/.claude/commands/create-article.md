# /create-article コマンド

指定テーマの記事を「ユニコ記事スタイル」で生成し、`.md` ファイルとして保存します。

## 手順

1. **参考記事の確認**
   - `参考記事/` フォルダ内のファイルをReadしてスタイルを把握する

2. **ヒアリング**（テーマが曖昧な場合のみ）
   - 記事のテーマ・キーワード
   - note向け / stand.fm向け / 両方
   - 熊取町要素の有無（デフォルト：含める）
   - 文字数の目安（デフォルト：3000文字以下）

3. **記事本文の生成**
   - ユニコ記事スタイルで構成する
   - 体験談 → 現状解説 → ツール紹介 → 地域活用 → FAQ → まとめ

4. **ハッシュタグの生成**
   - note用（最大15個）
   - stand.fm用（最大8個）

5. **stand.fm放送説明文の生成**
   - 150文字前後

6. **マークダウンでチャットに出力**
   - コードブロック（```markdown）で出力してコピペしやすくする

7. **`.md` ファイルとして保存**
   - 保存先：`C:\Users\motok\OneDrive\Desktop\Claude code\yuniko\[テーマ名]記事.md`
   - 記事本文・ハッシュタグ・stand.fm説明文を1ファイルにまとめる

## 使用例

```
/create-article AI動画について
/create-article 熊取町の移住メリット
/create-article Claude Codeの使い方
```
