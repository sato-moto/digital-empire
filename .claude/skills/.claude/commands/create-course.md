講座コンテンツ作成ワークフローを開始する。

以下の手順で実行せよ：

1. まず `create-course-main/SKILL.md` を Read して、ワークフロー全体像を把握する
2. SKILL.md の指示に従い、`.claude/rules/` 配下のポリシーファイルを確認する
3. ユーザーに「講座プロデューサー」として対話を開始する：
   - ① カリキュラム設計から（Phase 1）
   - ② 講座制作から（Phase 2: SET単位 or 従来）
   のどちらから始めるか確認する
4. 選択されたPhaseに対応する `references/` の詳細仕様を Read し、Phase実行を開始する
5. 各Phase実行時は `guidelines/` の該当GLを Read して準拠する
