# ClaudeCode 業務委譲分析 — media-assets

> **目的**: 各業務フェーズをClaudeCodeにどこまで委譲できるかを整理し、いつでも参照可能にする

---

## 凡例

| ランク | 意味 |
|--------|------|
| ✅ **完全自動化** | ClaudeCodeが単独で完遂可能 |
| 🔄 **半自動化** | ClaudeCodeが実行し、人間が最終確認のみ |
| ⚠️ **補助的** | ClaudeCodeが下準備、人間が仕上げる |
| ❌ **人間専任** | 現時点ではAI委譲が困難 |

## 総合分析

### 今すぐClaudeCodeに完全委譲できるタスク
1. アウトライン構成の自動生成
2. スライド構成（内容指示書）の自動生成
3. 台本・ナレーション原稿の自動生成
4. TTS音声ファイルの自動生成

### ワークフロー整備で委譲範囲を拡大できるタスク
1. スライド自動生成パイプライン（テンプレート + python-pptx）
2. アバター動画の自動生成パイプライン（API連携）
3. 動画の自動結合・書き出しパイプライン（FFmpeg）
4. 企画立案の半自動化（リサーチ→提案→承認フロー）

### 人間が引き続き担うべきタスク
1. **ライブ登壇・質疑応答**（物理的制約）
2. **最終品質判断**（ブランド整合性、教材としての適切さ）
3. **方向性の最終決定**（テーマ選定、企画承認）
