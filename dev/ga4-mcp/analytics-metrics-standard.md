## 総合評価の判定基準（5段階）

| 評価 | 判定基準 |
|------|---------|
| S：目標超過 | 主要指標の80%以上が「良好〜優秀」。フェーズ上位への移行を検討 |
| A：順調 | 主要指標の60%以上が「標準〜良好」。現戦略を継続 |
| B：概ね良好 | 強みと課題が混在。優先度の高い課題1〜2点に集中 |
| C：要改善 | 主要指標の50%以上が「要改善」。戦略の見直しが必要 |
| D：危機 | トラフィック・エンゲージメント共に悪化傾向。即時対応が必要 |

## 地域メディア特有の重点指標

1. **ローカルKW網羅率**：「熊取 ○○」形式のロングテールKWでインデックスされているか
2. **グルメ記事の鮮度**：飲食店情報は3ヶ月以内が原則。古い情報はCTRを下げる
3. **トップページ依存度**：トップページのPV比率が50%超の場合は内部リンク強化が急務
4. **新記事の立ち上がりCTR**：公開後7日で5%以上のCTRが出れば伸びる記事と判定
5. **特定ページ集中リスク**：上位3ページで全PVの70%超は集中リスクあり

## 総合評価の記述ルール（「アナリティクスまとめ最新」本文用）

- 細かい数値は記載しない（各日の子ページ参照）
- 現在のフェーズ判定と根拠を1〜2文で明示
- 「強み・弱み・最優先アクション」の3点構成
- 評価グレード（S/A/B/C/D）を明示
- 前回評価からの変化（改善・悪化・横ばい）を必ず言及
- 200文字以内を目安にコンパクトにまとめる
