エージェント & スキル構成
✍️
article-creator
記事生成サブエージェント
Agent
  • 参考記事フォルダを読み込みスタイルを把握
  • ユニコ記事スタイルで3000文字以下の記事本文を生成
  • note用（最大15個）・stand.fm用（最大8個）ハッシュタグ生成
  • stand.fm放送説明文を150文字前後で生成
  • .mdファイルとして所定のフォルダに保存
利用可能ツール：
Read Write Glob
🎬
video-creator
AI動画制作サブエージェント
Agent
  • 動画の目的・スタイルをユーザーと対話形式で確認
  • 台本の生成・管理（ユーザー提供素材を優先）
  • 音声クローンAPIでナレーション音声を生成
  • ナレーションに同期した字幕データを生成
  • Remotionで動画をコンポジット・レンダリング
  • 完成後にタイトル案5つと概要を報告
