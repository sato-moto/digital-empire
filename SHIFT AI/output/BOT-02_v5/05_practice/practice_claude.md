## 1. 収録の目的
教養編で学んだ設計思想（「AIに何を持たせるか」「ソースは一次情報を優先」「NotebookLM×Gems の役割分担」）を、**視聴者が画面の向こうで同時進行で手を動かせる**よう、NotebookLMの起動からソース追加・質問・範囲外検知・運用サイクルまでを1本で完走させる。視聴後に「自分の教材でやってみよう」と手元に戻れる状態を作ることがゴール。

## 2. 事前準備（撮影開始前にすべて完了）

### 2-1. 環境
- 画面解像度: 1920×1080（フルHD）
- Windows: タスクバー自動非表示 ON / 集中モード ON / デスクトップアイコン非表示
- Mac: Dock自動非表示 ON / メニューバー隠す / おやすみモード ON
- ブラウザ: Chromeシークレットウィンドウ（ブックマークバー・拡張機能アイコンが映らない）
- すべての通知（Slack / Teams / メール / カレンダー / SNS）停止
- マウスカーソル強調表示 ON（視認性のため）

### 2-2. 開いておくタブ・アプリ
1. **NotebookLM**（notebooklm.google.com）— Google アカウントでログイン済み、**ホーム画面（ノートブック一覧／新規作成ボタンが見える状態）**
2. 手元ローカルの **教材PDFフォルダ**（後でアップロード時にすぐ選べるよう、Finder / エクスプローラーで開いておく）

### 2-3. ダミーデータ（一貫ケース：高校2年生・数学・微分）
撮影前に以下3点のPDFを用意する。すべて**架空の生徒が作成したと見なせる内容**とし、実在校名・実在の先生名・実在のクラス名・実在生徒名は一切含めない：

1. `math2_note_bibun.pdf`（授業ノート／微分係数の定義と板書・10ページ前後）
2. `math2_print_bibun.pdf`（配布プリント／微分の練習問題と解法例・3ページ）
3. `math2_reference_bibun.pdf`（参考書の該当章PDF／微分章のみ抜粋・15ページ前後）

※ 実在の学校資料を使う場合は校名・個人名を**事前にマスク処理**（黒塗り／ぼかし）する。

### 2-4. 映してはいけないもの（NG事項）
- タスクバー／Dock／通知ポップアップ
- ブックマークバー／拡張機能アイコン
- ブラウザタブの履歴・他サイト
- 実在の学校名・先生名・クラス名・生徒名
- 成績情報・出席情報・連絡先
- Googleアカウントの名前欄に表示される実名（**撮影用サブアカウント推奨**）
- Gmailの通知（隣接タブに残さない）
- 他アプリのポップアップ

## 3. Phase構成（本編 — 詳細は同梱 CSV 参照）

| Phase | 内容 | 目安尺 |
|---|---|---|
| Opening | NotebookLMホーム画面を見せたまま、実践への切り出し＋前講座接続＋ゴール宣言 | 1分 |
| Phase 1 | 新規ノートブック作成／名前を「高2_数学_微分」に設定 | 1〜1.5分 |
| Phase 2 | 授業ノート→プリント→参考書PDFの順に3点アップロード／ソース読込完了確認 | 2〜2.5分 |
| Phase 3 | 「自分の解き方で説明」プロンプト入力／回答確認／引用スニペットを開いて根拠確認 | 2〜3分 |
| Phase 4 | あえて範囲外（積分の体積）を質問→「資料にありません」挙動を確認 | 1〜1.5分 |
| Phase 5 | 新単元ノート追加の手順を見せる（実際には1ソース追加のみ）＋不要ソース削除の手順（実際には操作のみで削除確認） | 1〜1.5分 |
| Ending | 完成画面を見せながら振り返り3点＋ねぎらい＋次SET（Gems編）への橋渡し | 1分 |

### Phase 3 補足（最重要）
- プロンプトはまるごとコピペできる形で CSV に記載。**手打ちせず、事前にテキストファイル（`prompts.txt`）からコピペする**動線を台本で見せる
- 回答が表示されたら必ず**引用スニペットをクリックして元ソースのハイライト箇所に飛ぶ**動作を入れる（これが視聴者に「根拠追跡」の強みを体感させる唯一の瞬間）

### Phase 4 補足
- 範囲外質問で「資料にありません」が返る画面は、**最低3秒静止**して映す（視聴者が文言を読めるように）
- 返答文が長文でも、要点部分にマウスカーソルを置いて読み上げる

---

## 4. 収録時の話し方ルール
- **全文「です・ます」調を徹底**（常体が1文でも混ざると講義の質感が崩れる）
- 一人称「私」、二人称「みなさん」
- 1文40字以内目安
- 操作の **直前** に「次に〜します」と予告 → 操作 → 「これで〜になりました」と結果確認、の3拍子
- プロンプト貼付け中の沈黙はOK。貼り終わってから1拍置いて話し出す
- 専門用語（ソース、ノートブック、引用スニペット）は初出時に1回だけ言い換える
