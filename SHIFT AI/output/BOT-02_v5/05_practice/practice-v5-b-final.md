# 実践動画 指示書：BOT-02_v5-b - 自分の学習Botを作ってみる

## 0. メタ情報
- **講座**: SHIFT AI講座 / BOT-02_v5-b / 自分の学習Botを作ってみる（実演）
- **対応章**: SET1「AIに教材を持たせる（NotebookLM編）」の実演パート（教養編 BOT-02_v5-a の続編）
- **想定尺**: 8〜12分（想定10分）
- **使用ツール**:
  - NotebookLM（PCブラウザ版・notebooklm.google.com）
  - Googleアカウント（**撮影用サブアカウント推奨**）
  - PDF教材4ファイル（下記2-3参照）
- **アウトプット（視聴後に手元に残るもの）**:
  1. 数学Ⅱ「微分」専用のNotebookLMノートブック1個
  2. 授業ノート2冊・配布プリント・参考書該当章PDFを読み込んだ学習環境
  3. 自分のノートの解き方で説明された微分の回答と引用スニペット履歴
  4. 新単元追加・不要ソース削除という運用サイクルの理解

## 1. 収録の目的
- みなさんが、NotebookLMに自分の教材を読み込ませて、範囲を絞った学習Botを立ち上げる流れを**一人で再現できる**状態にします。
- 特に、**ログイン → 新規ノートブック作成 → ソース追加 → 質問 → 引用確認 → 範囲外検知 → 運用サイクル**までを通しで見せます。
- 汎用AIに丸投げするのではなく、**先に教材を渡してから質問する**という発想を、手の動きとして定着させます。
- 今回入れるソースは**3カテゴリ／実ファイル4件**：①授業ノート（2冊）／②配布プリント／③参考書該当章。

## 2. 事前準備

### 2-1. 環境
- **画面解像度**: 1920×1080（フルHD）
- **デスクトップ**: タスクバー／Dock／通知／デスクトップアイコンが映らない
  - Windows: タスクバー自動非表示 ON
  - Mac: Dock自動非表示 ON、必要に応じてメニューバーも隠す
- **通知**: 集中モード／おやすみモード ON
- **ブラウザ**: Chromeシークレットウィンドウ推奨（ブックマークバー・拡張機能アイコン非表示）
- **ウィンドウサイズ**: ブラウザを最大化して録画範囲いっぱいに表示
- **マウスカーソル強調表示**: ON（視認性のため）
- **UI確認**: 収録直前にNotebookLMの最新UIを確認し、「新規作成」「追加」「削除」などのラベル差分を把握
- **推奨端末**: PC版で収録（資料一覧と対話欄を同時に見せやすいため）

### 2-2. 開いておくタブ・アプリ
- **NotebookLM**（notebooklm.google.com）— Googleアカウントでログイン済み、**ホーム画面**（ノートブック一覧／新規作成ボタンが見える状態）
- **プロンプト集テキストファイル**（`prompts.txt`）— **画面外の別ウィンドウ**でコピー可能な状態（本編では映さない）
- 教材PDF4ファイルを、ファイル選択ダイアログからすぐ選べる場所に保存
- 不要なタブはすべて閉じる
- Finder／エクスプローラーは本編で開かず、ファイル選択ダイアログ内で完結させる

### 2-3. ダミーデータ（一貫ケース：高校2年生・数学Ⅱ・微分）
今回の一貫ケースで固定：
- **主体**: 高校2年生・進学志望・自学自習型
- **科目**: 数学（高校数学Ⅱ）
- **単元**: 微分（微分係数の定義・導関数）
- **教材（3カテゴリ／4ファイル）**:
  - 授業ノート1: `数学II_授業ノート1_微分.pdf`
  - 授業ノート2: `数学II_授業ノート2_微分.pdf`
  - 配布プリント: `数学II_配布プリント_微分.pdf`
  - 参考書該当章: `数学II_参考書_微分_該当章.pdf`
  - **追加実演用（任意・Phase 5 で使用）**: `数学II_微分_追加ノート.pdf`（同じ微分単元内の追加ノート。1〜2ページの短いダミーPDF）
- PDFは文字が読める解像度で作成する
- 個人情報・実在校名は**事前にマスク処理**

#### プロンプト集（prompts.txt の中身）
以下を1つのテキストファイルに保存し、画面外のウィンドウで開いておく：

```
[PROMPT 1: 自分の解き方で説明]
アップロードした授業ノート2冊と配布プリントの情報だけを使って、
微分係数の定義を教えてください。
教科書以外の難しい定理（ロピタルの定理など）は絶対に使わないでください。
ノートに書かれている先生の板書の解説手順に沿って説明してください。

[PROMPT 2: 範囲外検知]
このノートの解き方を使って、積分を用いた体積の求め方を教えてください。
ノートに記載がない場合は「資料にありません」と明示してください。
一般知識で補わないでください。
```

- 別タブの中身、メール、メッセージアプリ、カレンダー通知

> **補足**

## 3. Phase構成（本編 — 詳細は同梱 `practice-v5-b.csv` 参照）

| Phase | 内容 | 目安尺 |
|---|---|---|
| Opening | NotebookLMホーム画面を見せたまま、実践への切り出し＋前講座思想の想起＋ゴール宣言 | 1分 |
| Phase 1 | 新規ノートブック作成／ノートブック名を「高2_数学_微分」に設定 | 1〜1.5分 |
| Phase 2 | 3カテゴリ4ファイルを順次アップロード／ソース読込完了確認 | 2〜2.5分 |
| Phase 3 | `prompts.txt` から PROMPT 1 をコピペ→回答確認→引用スニペットをクリックして根拠確認 | 2〜3分 |
| Phase 4 | PROMPT 2（範囲外検知）で「資料にありません」挙動を体感／汎用AIとの対比 | 1〜1.5分 |
| Phase 5 | `数学II_微分_追加ノート.pdf` を**実際に追加**／参考書PDFを**実際に削除**／運用サイクル解説 | 1〜1.5分 |
| Ending | 完成画面を映しながら振り返り3点＋宿題提示＋次SET（Gems編）への橋渡し | 1分 |

### Phase 0（Opening）補足
- 画面は常にNotebookLMのホーム画面を映したまま進行
- 歓迎演出（「ようこそ」「〇本目の講座へ」等）は入れない
- ここではログインはすでに済んでいる前提。まだの方へは短く案内のみ（動画停止の指示は入れない）

### Phase 1 補足
- 2段階認証が出る可能性を**短く**言及しておくと、編集でカットしやすい
- ノートブック名は**単元が分かる形**にする（「高2_数学_微分」）
- 「1ノートブック1単元くらいの粒度がおすすめ」と運用ポリシーを1文添える

### Phase 2 補足
- ファイル名はソース一覧に表示される前提で、「3カテゴリ4ファイル」を明示的に数える台本を入れる
- 読み込み待ちの間に「関係ない単元は入れない」という**教養編で学んだ判断基準**を想起させる

### Phase 3 補足（最重要）
- プロンプトは`prompts.txt`からコピペ（**手打ちしない**）
- 回答が表示されたら**必ず引用スニペットをクリックして元ソースのハイライト箇所に飛ぶ動作**を見せる
- 「プロンプトは事前にテキストファイルに用意しておくのがコツ」という運用TIPSを台本に織り込む

### Phase 4 補足
- 範囲外質問で「資料にありません」が返る画面は、**最低3秒は映したまま**待つ（視聴者が文言を読める時間を確保。音声は「このまま見てください」等で間を埋める）
- **「汎用AIだったら、ここで一般知識から体積公式を引っ張ってきて答えていたはず」** という対比台本を必ず入れる（範囲限定の価値を腹落ちさせる核）

### Phase 5 補足
- 追加実演：`数学II_微分_追加ノート.pdf`（同単元内の追加ノート）を実際にアップロード→ソース一覧に増える動き
- 削除実演：参考書PDFを実際に削除→ソース一覧が減る動き
- 「ノートブックは育てるもの」という運用メッセージで締める

### Phase 6（Ending）補足
- 振り返りは3点：①ノートブック作成／②ソース追加と質問／③範囲外検知と運用サイクル
- 宿題提示：「スマホで今使っている単元のノートを撮ってPDF化するところから始めてみて」
- 次SET接続：「NotebookLMは教材を持たせる。Gemsは先生の人格を持たせる」を再掲→姉妹SET「AIに人格を与える」へ
- 断定誇張（「このAIを使えば成績が上がる」等）は禁止

## 4. 収録時の話し方ルール（`practice_video_guideline.md §3-5` 準拠）
- **「です・ます」調を全文徹底**（常体「〜だ／〜である」は1文たりとも混ぜない）
- 一人称「**私**」、二人称「**みなさん**」（「あなた／視聴者の方」禁止）
- 1文は40字以内目安
- 各Phase冒頭で**次に何をするか予告** → 操作 → **何ができたか確認** の3拍子
- 入力中の沈黙はOK。入力後に1拍置いてから説明
- 操作説明と感情の言葉を混ぜる：「では入力します。……はい、出ましたね。ここがポイントです」
- 受講者への声かけを適度に：「ここは迷いやすいので、一緒に見ていきましょう」
- 初出の専門用語（ソース、ノートブック、引用スニペット）は1回だけ言い換え
- くだけた接続詞の置換：
  - 「だから」→「なので／だからこそ」
  - 「でも」→「ただ／ですが」
  - 「〜けど」→「〜ですが」
- 体言止めの連発禁止（3行以上連続禁止）
- **カメラ目線、会釈、歓迎演出は入れない**
