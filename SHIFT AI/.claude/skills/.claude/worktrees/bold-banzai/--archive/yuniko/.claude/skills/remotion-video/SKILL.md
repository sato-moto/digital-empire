# Remotion動画制作スキル

## 概要

このスキルは、Remotionフレームワークを使って動画を自動制作するための手順と知識を定義する。

## 前提条件

- Node.js がインストール済みであること
- Remotion がプロジェクトにセットアップ済みであること
- 音声API と 画像生成API のキーが設定済みであること

## 動画制作パイプライン

### 1. 台本生成
- ユーザーの指示に基づき、動画台本を生成する
- セクションごとに分割し、各セクションにナレーションテキストと画像指示を含める
- 出力形式: JSON（sections配列）

### 2. ナレーション生成
- 台本のナレーションテキストを音声APIに送信
- ユーザーの音声クローンが設定されている場合はそれを使用
- 出力: WAVまたはMP3ファイル

### 3. 字幕生成
- ナレーション音声のタイムスタンプに基づき字幕データを生成
- 出力形式: SRT または JSON

### 4. 画像生成
- 台本の画像指示に基づき、画像生成APIで画像を作成
- スタイルテンプレートに合わせたプロンプトを構築
- 出力: PNG ファイル

### 5. Remotionコンポジション
- 全素材（音声、字幕、画像、BGM）をRemotionコンポジションに統合
- スタイルテンプレートに基づいたアニメーション・トランジションを適用
- プレビューで確認後、最終レンダリング

### 6. レンダリング
- `npx remotion render` でMP4を出力
- 縦動画: 1080x1920、横動画: 1920x1080
