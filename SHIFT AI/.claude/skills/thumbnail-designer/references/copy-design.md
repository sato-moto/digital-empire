# コピー設計ガイド

Phase 3 専用。企画情報 + ターゲット心理 → サブタイトル5案を生成する。
講座名を補強し、ベネフィットを伝える1行を設計する。

---

## 大原則

企画テキストの直訳は絶対にNG。ベネフィットドリブンで再設計する。

※ テキスト階層・講座名保護・句読点のルールは **.claude/rules/text-hierarchy.md** を参照

---

## STEP 1: ターゲット心理の掘り下げ

コピーを出す前に、以下の3つを言語化する。これを飛ばさない。

| 問い | 例 |
|------|-----|
| ターゲットが抱えている不安は何か？ | 「自分には売れるスキルがない」「始めても続かない」 |
| ターゲットが言い訳にしていることは何か？ | 「未経験だから無理」「センスがないから」 |
| ターゲットが本当に欲しい状態は何か？ | 「インスタで安定して売れる仕組みを持ちたい」 |

この3つが明確になってから、それを**打ち崩す・代弁する・叶える**コピーを生成する。

---

## STEP 2: 煽りワード変換

「月100万」「稼ぐ」「独立」「副業」「不労所得」などの煽りワードは、画像生成AIの学習データにおいて低品質な広告・クリック誘発型バナーと結びついており、そのまま使うとデザインクオリティが著しく下がる。

ワードを丸ごと消すのではなく、同じ訴求力を持つ別の表現に置き換える。

| 煽りワード | NG例 | OK例 | ポイント |
|-----------|------|------|---------|
| 月100万稼ぐ | 月100万稼ぐ方法 | 月収を3倍にした設計図 | 数字は残す。「稼ぐ」を「3倍」に変換 |
| 独立・副業 | 副業で稼ぐ | 会社依存ゼロの仕事術 | 状態を動詞で表現 |
| 稼げる・儲かる | 儲かるAIコンサル | 指名が止まらないAIコンサル | 「儲かる」を結果の動詞に置き換え |

---

## STEP 3: サブタイトル（5案）

講座名だけでは伝わらないベネフィット・具体性・参入障壁の除去を1行で補う。

| 考え方 | 説明 |
|--------|------|
| フック | 「自分ごと」と思わせる一撃 |
| ヒーロー語 | 感情を動かす最大級の言葉 |
| 具体性 | ツール名・数字・固有名詞で対象を明確に。「商品」「サービス」など曖昧な単語は具体的な言葉に置き換える |
| ベネフィット | 「これを得られる」が伝わる |
| 内容の明示 | 「何を学ぶか」または「何ができるようになるか」を含める |
| 参入障壁の除去 | 「自分でもできる」と思わせる。スキル・経験・時間・お金など、ターゲットが感じている不安を名指しして取り除く |

**判定基準**: 講座名 + サブタイトルを合わせて見て、内容を知らない人が「何の講座か」を説明できるか？できなければ抽象的すぎる。

**長さの警告**: サブタイトルが20文字を超える場合、サムネイル上でバランスが崩れるリスクがある。その案には⚠️を付けて「長すぎる可能性あり」と明記し、短縮案を併記する。

```
【サブタイトル案】
案1: ○○○○○（パターン名）
案2: ○○○○○
案3: ○○○○○
案4: ○○○○○
案5: ○○○○○
```

**出力時の強調ルール**: 各案の中で最も伝えたい重要フレーズを**太字**にして明示する。このフレーズはPhase 4のプロンプトに「『〇〇』をアクセントカラーで強調すること」と反映させる。例: `**数学力不要**　問いを立てる力だけで始められる` → プロンプトに「『数学力不要』をアクセントカラーで強調すること」と追記

→ ユーザーにサブタイトルを選んでもらう。
