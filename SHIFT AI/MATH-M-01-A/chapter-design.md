# Step 4-T3: 章ロール設計 — MATH-M-01-A

## 章構成

### Ch0：冒頭あいさつ
- 章タイトル：この動画で手に入るもの
- 章ロール：興味喚起（フック）
- 章ゴール：受講者が「この動画を最後まで見たい」と感じている状態になる

### Ch1：AIに「なぜ？」を聞いて計算の仕組みをつかむ
- 章タイトル：AIに「なぜ？」を聞いて計算の仕組みをつかむ
- 章ロール：認識転換＋概念理解
- 章ゴール：受講者が「計算ルールを暗記するのではなく、仕組みをAIに問う」という学習姿勢を説明でき、3種のプロンプトパターンを使い分けられる状態になる

### Ch2：実演・計算編（録画映像を差し込む）
- 章タイトル：実演・計算編
- 章ロール：実行体験（デモ）
- 章ゴール：【空白チャプター】録画映像差込。内容は制作者が後から追加
- ※尺目安：10〜20分

### Ch3：NotebookLMで自分専用の数学コーチを作る
- 章タイトル：NotebookLMで自分専用の数学コーチを作る
- 章ロール：応用展開（ツール連携による自走環境構築）
- 章ゴール：受講者が「学習メモや教材PDFをNotebookLMにアップロードし、自分専用の数学コーチとして日常的に使える」状態になる

### Ch4：まとめ＆激励エール
- 章タイトル：ここからはあなたの番
- 章ロール：統合定着＋自走スイッチ起動
- 章ゴール：受講者が「学んだ内容を整理し、今日から自分でAIに問いかけて学習を始める意志を持つ」状態になる

## 心理順序マッピング

| 心理順序 | 対応章 |
|----------|--------|
| 興味 | Ch0（フック＋ベネフィット提示） |
| 納得 | Ch1（認識転換＋プロンプト3種の概念理解） |
| 実行 | Ch2（実演デモ）→ Ch3（ツール連携で環境構築） |
| 自信 | Ch4（まとめ＋エールで自走スイッチ起動） |

## NGチェック（7項目）

- [x] 各章ゴールが講座ゴールに貢献している → 全章が「仕組みを問い自走する」に収束
- [x] 各章ロールが明確に定義されている → 興味喚起/認識転換/実行体験/応用展開/統合定着
- [x] 章同士に内容重複がない → Ch1=考え方、Ch2=実演、Ch3=ツール環境、Ch4=まとめ
- [x] ゴール達成に必要な章が欠けていない → マインドセット→プロンプト→実演→環境構築→自走
- [x] 難易度順序が逆転していない → 概念→実践→応用→統合の順
- [x] 実行フェーズの章が存在する → Ch2（実演）+ Ch3（ツール連携）
- [x] 導入・認識フェーズの章が存在する → Ch0（興味）+ Ch1（認識転換）
